# KODO — Code Snippet Manager

Save, organize, and drag-and-drop code snippets directly in your IDE.

## Features

- **📁 Folder Organization** — Nest your snippets in collapsible folders
- **🏷️ Tag System** — Color-coded tags for cross-cutting categorization  
- **🔍 Instant Search** — Fuzzy search across snippet names, code, and descriptions
- **✨ Syntax Highlighting** — Previews with language-aware highlighting
- **🎯 Drag & Drop** — Drag snippets from the sidebar into your editor
- **📋 Insert at Cursor** — One-click insertion with template variable support
- **🖱️ Right-Click Save** — Select code → right-click → "Save to KODO"
- **📤 Export / Import** — Back up and share your collection as JSON
- **🎨 Theme Aware** — Automatically matches your IDE theme (dark, light, high contrast)

## Template Variables

Use VS Code snippet syntax for dynamic placeholders:

```
const ${1:name} = ${2:value};
```

When inserted, tab through placeholders to fill them in.

## Usage

1. **Save a snippet**: Select code in the editor → Right-click → *Save to KODO*
2. **Browse snippets**: Click the KODO icon in the activity bar
3. **Insert a snippet**: Click *Insert* on any snippet card, or drag it into the editor
4. **Organize**: Create folders and tags to keep your collection tidy
5. **Export/Import**: Use the command palette → *KODO: Export Snippets* / *KODO: Import Snippets*

## Compatibility

Works in **VS Code**, **Cursor**, and **Antigravity**.
